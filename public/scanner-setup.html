02

PC 연결 설정 (2.4GHz 모드)

Windows PC에서 2.4GHz 동글로 연결하기

권장: PC에서는 2.4GHz 연결이 가장 안정적입니다. 드라이버 설치 없이 플러그앤플레이로 사용 가능합니다.

2.4GHz 페어링 설정 바코드

아래 바코드를 순서대로 스캔하세요

1단계

설정 모드 진입

Enter Setting Mode

2단계

2.4G 무선 모드

2.4G Wireless Mode

3단계

동글 페어링

Pair with Dongle

4단계

설정 모드 종료

Exit Setting Mode

A 설정 순서

  1. 1 스캐너 전원을 켭니다 (버튼 길게 누름)
  2. 2 "설정 모드 진입" 바코드를 스캔합니다
  3. 3 "2.4G 무선 모드" 바코드를 스캔합니다
  4. 4 "동글 페어링" 바코드를 스캔합니다
    (왼쪽 파란 LED가 깜빡임)
  5. 5 USB 동글을 PC에 연결합니다
    ("삐" 소리가 나면 페어링 성공)
  6. 6 "설정 모드 종료" 바코드를 스캔합니다

! 주의사항

  • 페어링 모드에서 1분 내 동글을 연결하지 않으면 자동으로 취소됩니다
  • USB 케이블이 연결된 상태에서는 유선 모드로 동작합니다
  • 페어링 성공 시 오른쪽 파란 LED가 계속 켜져 있습니다
  • 한 번 페어링하면 재시작 후에도 자동 연결됩니다
03

모바일/태블릿 연결 (블루투스 HID 모드)

Android/iOS 기기에서 블루투스로 연결하기

블루투스 HID 모드: 스캐너가 블루투스 키보드처럼 동작하여 스캔한 바코드가 텍스트로 입력됩니다.

방법 1: 간편 연결 (버튼 방식)

  1. 1

    스캐너 전원 버튼을 8초간 길게 누릅니다

    → 블루투스 HID 페어링 모드로 진입

  2. 2

    모바일 기기에서 블루투스 설정을 열고 "Barcode Scanner HID"를 찾습니다

  3. 3

    "Barcode Scanner HID"를 선택하여 연결합니다

    → "삐" 소리가 나면 연결 성공

방법 2: 바코드 스캔 연결

1단계
[설정 모드 진입]

Enter Setting Mode

2단계
[블루투스 HID 모드]

Bluetooth HID Mode

3단계
[설정 모드 종료]

Exit Setting Mode

위 바코드 스캔 후 모바일 기기의 블루투스 설정에서 "Barcode Scanner HID"를 검색하여 연결합니다.

iOS (iPhone/iPad)

  • 설정 → 블루투스 → "Barcode Scanner HID" 연결
  • 연결 후 키보드가 자동으로 숨겨질 수 있음
  • ERP 웹에서 입력 필드 터치 후 스캔

Android

  • 설정 → 연결된 기기 → 새 기기 페어링
  • Android 3.0 이상 필요
  • 일부 기기는 "입력 기기"로 표시됨

자주 묻는 질문 (FAQ)

스캐너가 연결되었는데 ERP에 입력이 안 됩니다
  • • 바코드 입력창에 커서가 있는지 확인하세요 (클릭하여 포커스)
  • • 스캐너가 "저장 모드"가 아닌 "일반 모드"인지 확인하세요
  • • USB 케이블이 연결된 상태라면 분리 후 무선으로 테스트하세요
블루투스 연결이 자주 끊깁니다
  • • 스캐너와 기기 사이 거리가 30m 이내인지 확인하세요
  • • 절전 모드 시간을 "절전 안함"으로 설정하세요
  • • 다른 블루투스 기기와의 간섭이 있을 수 있습니다
  • • PC에서는 2.4GHz 동글 연결을 권장합니다
스캔 후 엔터(Enter)가 입력되지 않습니다
  • • "설정 모드 진입" → "Add CR" 바코드 스캔 → "설정 모드 종료"
  • • CR = Carriage Return = Enter 키
  • • 기본값은 CR+LF가 추가되어 있습니다
이전에 페어링한 기기에서 재연결이 안 됩니다
  • • 기기의 블루투스 설정에서 "Barcode Scanner HID"를 먼저 삭제(해제)하세요
  • • 스캐너를 재시작하고 다시 페어링 과정을 진행하세요
  • • 다른 기기와 먼저 페어링된 경우 해당 기기에서도 삭제 필요
