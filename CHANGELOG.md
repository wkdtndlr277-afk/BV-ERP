# 변경 이력 (Changelog)

## (주)본비반트 HACCP 통합관리시스템

---

## [1.1.0] - 2026-01-29

### 🔐 사용자 인증 및 접근 권한 시스템

#### ✅ 신규 기능
- **회원가입**: 아이디, 비밀번호, 이름으로 가입 신청
- **로그인/로그아웃**: 세션 기반 인증
- **관리자 승인 시스템**: 가입 신청 후 관리자 승인 필요
- **권한 관리**: 
  - `super_admin`: 최고 관리자 (사용자 관리 가능)
  - `admin`: 관리자 (모든 데이터 수정/삭제 가능)
  - `user`: 일반 사용자 (조회 및 기본 입력만 가능)

#### 👥 사용자 관리 (관리자 모드)
- 승인 대기 사용자 목록
- 사용자 승인/거부
- 권한 변경 (user → admin → super_admin)
- 사용자 삭제
- 사용자 목록 조회

#### 🔒 접근 통제
- 비승인 사용자 접근 차단
- 로그인 필요 화면 자동 리다이렉트
- 관리자 기능 권한 체크

#### 📝 데이터베이스
- `Users` 테이블 추가 (id, username, password, name, role, approval_status, created_at)
- `User_Sessions` 테이블 추가 (세션 관리)
- 기본 super_admin 계정: admin / admin1234

## [1.0.0] - 2026-01-29

### 🎉 최초 릴리즈

#### ✅ 핵심 기능
- **대시보드**: 실시간 현황 (안전재고 미만, 유통기한 임박, 품질 KPI 알림)
- **입고 등록**: LOT 자동 생성, 신규 원료 등록, 원료 일괄 업로드 (CSV)
- **사용량 입력**: FEFO 자동 적용 (선입선출)
- **출고 등록**: 제품 출고 관리
- **제품 재고 등록**: 검색 기반 수량 수기 입력

#### 📊 재고/수불 관리
- **재고 현황**: 품목별 현재고, 안전재고 비교
- **수불 통합검색**: 기간별, 품목별, 유형별 검색
- **LOT 이력**: LOT별 입출고 추적
- **일별 수불부**: 일자별 입고/사용/출고/조정 내역
- **월별 수불부**: 월초재고, 입출고, 월말재고

#### 🧪 품질 관리
- **품질 KPI**: 일일 품질 지표 등록/관리, 적합/부적합 판정
- **반제품 공정품질**: 반죽온도, pH, 습도, 발효시간 기록 및 자동 판정
- **반죽 기준 마스터**: 8종 반죽 기준값 관리

#### 📦 기준정보 관리
- **품목 관리**: 원료/제품 마스터, 일괄 업로드
- **거래처 관리**: 입고/출고/양방향 거래처, 검색 필터
- **제품 현황 관리**: 제품사진, 품목제조보고서, 바코드, 소비기한, 보관방법, 판매처

#### 🔐 관리자 모드
- 비밀번호 인증 (기본: admin1234)
- 입고 데이터 수정/삭제
- 트랜잭션 수정/삭제 (재고 자동 연동)
- 재고 강제 조정
- 전체 재계산
- 활동 로그 기록

#### 📤 데이터 내보내기
- 모든 보고서 엑셀 다운로드 (CSV)
- 모든 보고서 출력 기능

#### 🛠 기술 스택
- Frontend: Vanilla JS + TailwindCSS + FontAwesome
- Backend: Hono (Cloudflare Workers)
- Database: Cloudflare D1 (SQLite)
- Deploy: Cloudflare Pages
